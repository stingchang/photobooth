## Picoria
>The goal of this project is to create an online platform to allow users to store photos and connect friends.

#### Tech Stack aimed to use
* Java 8
* Spring Boot
* Jersey RESTFul API
* Hibernate ORM
* Front End
    * CSS
    * HTML
    * JavaScript
    * jQuery
    * Bootstrap
* Cloud: Amazon Web Service
    * EC2   - Ubuntu Web Server
    * RDS   - MySQL
    * S3    - Assets storage
